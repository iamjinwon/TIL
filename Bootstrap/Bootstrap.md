## Bootstrap

- **부트스트랩(Bootstrap)**은 웹사이트를 쉽게 만들 수 있게 도와주는 CSS, JS **프레임워크**이다. 하나의 코드로 휴대폰, 태블릿, 데스크탑까지 다양한 기기에서 작동하게 만들 수 있으며, 다양한 기능을 제공하여 사용자가 쉽게 웹사이트를 제작, 유지, 보수할 수 있도록 도와준다. 출처: 위키백과
